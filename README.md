# BnB Price Calculator

A simple web dashboard for Airbnb and Booking hosts to calculate optimal pricing. Quickly see what you earn from what guests pay and compare prices between Airbnb and Booking in one place.

---

## Features

- **Instant conversion:** See what you earn from the price guests pay.
- **Multi-platform comparison:** Check Airbnb vs Booking pricing side by side.
- **Customizable rules:** Adjust for taxes, fees, or special pricing scenarios.
